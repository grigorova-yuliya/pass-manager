# Password Manager

A simple yet secure command-line password manager built in Python with encryption support.

## Features

 **Key Features:**
- Master password protection for all stored passwords
- Secure encryption using Fernet (symmetric encryption)
- Local JSON-based storage with restricted file permissions
- Add new password entries
- Retrieve stored passwords
- List all stored services
- Delete password entries
- Simple and intuitive CLI interface

## Installation

### Prerequisites
- Python 3.7 or higher
- pip (Python package manager)

### Setup

1. Clone or download this repository
2. Navigate to the project directory
3. Install dependencies:

### bash
pip install -r requirements.txt
